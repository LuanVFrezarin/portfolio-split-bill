# Divisor de Contas (Racha 2.0)

Aplicativo de exemplo: frontend em React + Vite e backend em Node.js + Express com Socket.io.

Como rodar em desenvolvimento:

Backend:
```
cd backend
npm install
npm run dev
```

Frontend:
```
cd frontend
npm install
npm run dev
```

Descrição: versão simplificada que implementa sessões com QR Code, sincronia em tempo real via Socket.io, divisão automática e histórico salvo em arquivo JSON. PDF export básico incluído.
